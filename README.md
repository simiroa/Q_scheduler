# 퀀텀 스케줄러 (Quantum Scheduler) - 사용자 매뉴얼

퀀텀 스케줄러는 로컬 네트워크를 통해 여러 기기(PC, 스마트폰, 태블릿)에서 동시에 일정을 관리할 수 있는 강력한 일정 관리 프로그램입니다.

## 📥 1. 설치 방법

1.  `dist` 폴더 안에 있는 **`QuantumScheduler_Setup.exe`** 파일을 실행하여 설치를 진행합니다.
2.  설치가 완료되면 **'Quantum Scheduler'**가 자동으로 실행됩니다.
3.  바탕화면의 바로가기를 통해 언제든 다시 실행할 수 있습니다.

---

## 🚀 2. 실행 및 사용법

### 💻 서버 PC (프로그램이 설치된 컴퓨터)
*   프로그램을 실행하면 **시스템 트레이(화면 우측 하단 시계 옆)**에 파란색 원 모양의 아이콘이 나타납니다.
*   **브라우저 자동 실행:** 프로그램 시작 시 자동으로 웹 브라우저가 열리며 스케줄러 화면(`http://localhost:8088`)이 나타납니다.
*   **트레이 메뉴 기능:**
    *   **Open Browser:** 내 PC에서 스케줄러 열기
    *   **Copy Network Address:** 다른 기기에서 접속할 수 있는 주소 복사
    *   **Network Info:** 현재 연결 정보 확인
    *   **Quit:** 서버 종료

### 📱 다른 기기에서 접속하기 (스마트폰, 다른 PC)
1.  서버 PC의 트레이 아이콘을 우클릭하여 **[Copy Network Address]**를 클릭합니다.
    *   예: `http://192.168.0.93:8088`
2.  스마트폰이나 다른 컴퓨터의 웹 브라우저(크롬, 사파리 등)를 엽니다.
3.  주소창에 복사한 주소를 입력하고 접속합니다.
4.  이제 PC와 스마트폰에서 **동시에** 일정을 수정하고 확인할 수 있습니다.

---

## ⚠️ 3. 접속이 안 될 때 (중요!)

다른 기기에서 로딩이 계속되거나 접속이 안 된다면 다음을 확인해주세요.

### 1) 같은 와이파이(공유기) 사용 필수
*   서버 PC와 스마트폰이 **반드시 같은 공유기(Wi-Fi)**에 연결되어 있어야 합니다.
*   스마트폰이 LTE/5G 데이터 모드라면 와이파이로 전환해주세요.

### 2) 방화벽 설정 확인 (가장 흔한 원인)
윈도우가 현재 네트워크를 **'공용 네트워크'**로 인식하면 외부 접속을 차단합니다. **'개인 네트워크'**로 변경해야 합니다.

1.  `[설정]` -> `[네트워크 및 인터넷]` -> `[이더넷]` (또는 Wi-Fi) 선택
2.  네트워크 프로필 유형을 **[개인] (Private)**으로 선택
3.  이제 다시 접속을 시도해 보세요.

---

## 🛠️ 4. 주요 기능

*   **자동 포트 감지:** 기본 포트(8088)를 다른 프로그램이 사용 중이면, 자동으로 8089, 8090... 순서로 빈 포트를 찾아 실행합니다. 충돌 걱정 없이 사용하세요.
*   **데이터 자동 저장:** 저장 버튼을 누를 때마다 설치된 폴더 내 `server/list`에 안전하게 저장됩니다.
*   **동시 접속:** 여러 기기에서 동시에 접속하여 내용을 확인할 수 있습니다. (※ 수정은 마지막에 저장한 사람의 내용이 반영됩니다.)

## 📂 5. 프로젝트 관리
*   **새 프로젝트 추가:** 왼쪽 사이드바 하단의 `Add Project` 버튼 클릭
*   **프로젝트 삭제:** 프로젝트 목록의 휴지통 아이콘 클릭 (삭제 시 복구 불가하니 주의하세요!)
*   **데이터 위치:** 설치 폴더 내 `server/list` 폴더에 `.json` 파일로 저장됩니다. 백업이 필요하면 이 폴더만 복사해두세요.
